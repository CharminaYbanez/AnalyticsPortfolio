\# Project Folder Structure



\## data\_raw

Original datasets.  

Never modify these.  

Always keep raw data immutable.



\## data\_clean

Processed, cleaned, or transformed datasets.  

Safe for analysis and modeling.



\## notebooks

All Jupyter notebooks.  

Exploration, testing, analysis, documentation.



\## scripts

Reusable Python scripts (.py).  

Functions, utilities, automation.



\## outputs

Charts, reports, model files, generated artifacts.



\## docs

Documentation, notes, templates, shortcuts, project rules.
