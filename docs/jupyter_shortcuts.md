\# JupyterLab Essential Shortcuts (Windows)



A fast-reference guide for efficient notebook navigation, editing, and execution.



\---



\## Running Cells

\- \*\*Shift + Enter\*\* — Run cell, move to next  

\- \*\*Ctrl + Enter\*\* — Run cell, stay in place  

\- \*\*Alt + Enter\*\* — Run cell, insert new cell below  



\---



\## Modes

\### Edit Mode (type inside cell)

\- \*\*Enter\*\* — Switch to Edit Mode  



\### Command Mode (blue border)

\- \*\*Esc\*\* — Switch to Command Mode  



\---



\## Cell Creation \& Deletion (Command Mode)

\- \*\*A\*\* — Insert cell above  

\- \*\*B\*\* — Insert cell below  

\- \*\*D, D\*\* — Delete cell  

\- \*\*Z\*\* — Undo cell deletion  



\---



\## Cell Type

\- \*\*Y\*\* — Change to Code cell  

\- \*\*M\*\* — Change to Markdown cell  



\---



\## Navigation

\- \*\*Up / Down Arrow\*\* — Move between cells  

\- \*\*Shift + Up/Down\*\* — Select multiple cells  



\---



\## Cell Operations

\- \*\*Shift + M\*\* — Merge selected cells  

\- \*\*Ctrl + Shift + -\*\* — Split cell at cursor  



\---



\## File \& Notebook

\- \*\*Ctrl + S\*\* — Save notebook  

\- \*\*Ctrl + Shift + S\*\* — Save all open documents  

\- \*\*Ctrl + P\*\* — Command palette (search commands)  

\- \*\*Ctrl + B\*\* — Toggle left sidebar (file browser)  



\---



\## Text Editing (Edit Mode)

\- \*\*Ctrl + A\*\* — Select all  

\- \*\*Ctrl + Z\*\* — Undo  

\- \*\*Ctrl + Y\*\* — Redo  

\- \*\*Ctrl + /\*\* — Toggle comment  



\---



\## Interrupt / Restart Kernel (Command Mode)

\- \*\*I, I\*\* — Interrupt kernel  

\- \*\*0, 0\*\* — Restart kernel  



\---



\# Recommended Daily Workflow

1\. \*\*Esc\*\* → Command Mode  

2\. \*\*B\*\* → New cell  

3\. \*\*Enter\*\* → Edit Mode  

4\. Write code  

5\. \*\*Shift + Enter\*\* → Run  

6\. \*\*Ctrl + S\*\* → Save  



This rhythm becomes second nature.
